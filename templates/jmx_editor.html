JMX Editor Help

Getting Started

  • Drag components from the palette to the canvas
  • Click on components to select and edit their properties
  • Nest components by dropping them inside containers
  • Use the Load menu to import existing JMX files or samples
  • Use the Validate button to check your test plan
  • Generate JMX to download the final test file

Loading Test Plans

  • Upload JMX File: Import existing JMeter test plans
  • Load Sample: Start with a pre-built example
  • Load JSON: Import previously saved test plan files

Component Types

  • Test Plan: Root container for all elements
  • Thread Group: Defines user simulation
  • HTTP Request: Makes web requests
  • Listeners: Collect and display results
  • Config Elements: Provide configuration data
